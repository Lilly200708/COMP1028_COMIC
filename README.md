## COMP1028_COMIC

### Task 1: Comis Generation
You are required to use any AI tools to generate a comic for each of the following four sorting algorithms to illustrate their concepts:
>1. Insertion Sort
>2. Selection Sort
>3. Merge Sort
>4. Bubble Sort
The number of panels for each comic must be 8, 12, or 16.

File Submission Format: COMP1028_GroupName_Comic.jpg

### Task 2: AI Log Book 
You are required to record all the prompts you use and the responses you receive when generating the comic for each algorithm.
>Part 1: State the name of the AI tool used and justify your selection.

>Part 2: Prompt, AI response, and your reflection

**Guideline:**

Do not just show the final, perfect prompt. Instead, show the "journey" by recording the first 5 prompts and the last prompt used for each of the four algorithms.

File Submission Format: COMP1028_GroupName_Log.pdf
